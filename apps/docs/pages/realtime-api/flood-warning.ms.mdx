## Endpoint API Amaran Banjir

```http
GET https://api.data.gov.my/flood-warning
```

### Sumber Data Amaran Banjir

API Amaran Banjir ini menyediakan maklumat dalam masa nyata tentang tahap air dan amaran banjir di stesen pemantauan di seluruh negara.
Data ini telah dikumpul dan disediankan oleh Jabatan Pengairan dan Saliran.
Anda juga boleh mengakses semua data API Amaran Banjir ini di pautan [Pautan Rasmi Awam Infobanjir](https://publicinfobanjir.water.gov.my/main/).

### Kekerapan Kemaskini Data

Data API Amaran Banjir ini dikemaskini pada setiap **15 minit**.

## Format Permintaan & Respons

Untuk menyaring dan meminta data, gunakan operasi penyaringan standard dalam [Permintaan Pertanyaan](/request-query). Untuk maklumat tentang bagaimana API ini memberi respons kepada permintaan anda, rujuk bahagian [Format Respons](/response-format).

### Contoh Permintaan

Untuk mendapatkan data dari API Amaran Banjir, gunakan perintah `curl` berikut. Buka terminal anda dan masukkan perintah ini:

```curl
curl --location 'http://api.data.gov.my/flood-warning?limit=3'
```
