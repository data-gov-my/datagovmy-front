# Pengenalan

API Terbuka yang dibina oleh kerajaan Malaysia ialah API RESTful yang direka untuk menyediakan akses data yang telus kepada semua rakyat,
membolehkan akses kepada set data penting daripada pelbagai agensi kerajaan. Anda boleh mencari data tentang pengangkutan, cuaca, pilihan raya dan banyak lagi.

Menggunakan API Terbuka ini, pengguna boleh mendapatkan semula data dengan cara yang mudah dan mudah difahami.
API direka bentuk untuk mesra pengguna dan halaman dokumentasi telah dibuat untuk membantu pengguna memahami cara menggunakan API dengan cekap.
Pengguna boleh mengharapkan untuk menerima data yang boleh dipercayai dan tepat melalui API ini, menjadikannya sumber penting bagi sesiapa sahaja yang ingin mengakses data di Malaysia.
