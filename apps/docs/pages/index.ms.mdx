## Dokumentasi OpenAPI

Selamat datang ke dokumentasi OpenAPI portal rasmi Data Terbuka Malaysia. 
Panduan komprehensif ini dibangunkan bagi memberi maklumat yang diperlukan untuk menggunakan OpenAPI. 
Dokumentasi ini akan berfungsi sebagai panduan kepada pembangun sistem, saintis data atau orang awam yang berminat untuk mengakses data yang telus bagi memanfaatkan OpenAPI.

## Apakah OpenAPI Ini?

OpenAPI yang dibina oleh kerajaan Malaysia adalah API berdasarkan prinsip REST yang menggunakan rangkaian Django.
API ini telah dibangunkan dengan matlamat untuk menyediakan akses data yang telus kepada orang awam dengan matlamat-matlamat berikut:

- **Akses Data yang Telus:** API ini adalah mesra pengguna yang membolehkan capaian kepada pelbagai set data, menyokong pembuatan keputusan yang telus dan berinformasi.
- **Kemudahan Penggunaan:** API ini direka untuk mudah difahami dan memastikan pengguna boleh mendapatkan data dengan efisien.
- **Pelbagai Set Data:** OpenAPI menawarkan capaian kepada pelbagai data, membenarkan pengguna meneroka dan menganalisis kepelbagaian aspek masyarakat Malaysia.
