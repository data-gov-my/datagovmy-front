## Apakah OpenAPI Ini?

OpenAPI yang dibina oleh kerajaan Malaysia adalah API berdasarkan prinsip REST yang menggunakan rangkaian Django.
API ini telah dibangunkan dengan matlamat untuk menyediakan akses data yang telus kepada semua warganegara, dengan matlamat-matlamat berikut:

- **Akses Data yang Telus:** API ini menyediakan cara yang mesra pengguna untuk mengakses pelbagai set data, mempromosikan telus dan membolehkan pembuatan keputusan yang berinformasi.
- **Kemudahan Penggunaan:** API ini direka untuk mudah difahami dan difahami, memastikan pengguna boleh mendapatkan data dengan cekap tanpa kerumitan yang tidak perlu.
- **Pelbagai Set Data:** OpenAPI menawarkan akses kepada pelbagai data, membenarkan pengguna meneroka dan menganalisis pelbagai aspek masyarakat Malaysia.
