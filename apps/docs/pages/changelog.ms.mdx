## 1.2.0 (2023-11-06)

### Pelancaran API GTFS Statik

- Pelancaran [API GTFS Statik](ms/realtime-api/gtfs-static) dengan data dari myBAS Johor Bahru, KTMB, dan Prasarana dengan 'endpoints' berikut:
  - `gtfs-static/mybas-johor` untuk muat turun fail ZIP GTFS myBAS.
  - `gtfs-static/ktmb` untuk muat turun fail ZIP GTFS KTMB.
  - `gtfs-static/prasarana?category=<kategori>` untuk muat turun fail ZIP GTFS Prasarana.

## 1.1.0 (2023-10-09)

### Pelancaran API Cuaca

- Pelancaran [Cuaca API](ms/realtime-api/weather) dengan data dari [MET Malaysia](https://www.met.gov.my/) dengan 'endpoints' berikut:
  - `/weather/forecast` untuk mengakses data ramalan umum 7 hari.
  - `/weather/warning` untuk mengakses data masa nyata amaran cuaca.
  - `/weather/warning/earthquake` untuk mengakses data amaran gempa bumi.

## 1.0.0 (2023-09-13)

### Pelancaran Awal

- Pelancaran OpenAPI Malaysia dengan 'endpoints' berikut:
  - `/data-catalogue` untuk mengakses katalog data.
  - `/opendosm` untuk mengakses katalog data OpenDOSM.
  - `/flood-warning` untuk mengakses data amaran banjir secara waktu sebenar.
