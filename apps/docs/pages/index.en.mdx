# Introduction

The Open API built by the Malaysian government is a RESTful API designed to provide transparent data access to all citizens,
enabling access to crucial datasets from a plaethora of government agencies. You can find data about transportation, weather, elections and many more.

Using this Open API, users can retrieve data in a straightforward and easy-to-understand manner.
The API is designed to be user-friendly, and the documentation page has been created to help users understand how to use the API efficiently.
Users can expect to receive reliable and accurate data through this API, making it an essential resource for anyone looking to access data in Malaysia.
